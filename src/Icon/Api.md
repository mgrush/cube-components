### Icon

| 属性          | 说明                                           | 默认值 |
| ------------- | ---------------------------------------------- | ------ |
| name         | 图标的类型，对应Font Awesome的图标名称         | -      |
| size          | 图标尺寸的大小，支持:small、medium、large      | medium |
| color |图标显示的颜色，只针对字体图标有效|white|
| className | 样式类 | - |
| url           | 网络图标资源链接，优先由type决定图标显示的类型 | -      |
| onClick       | 用户点击时的响应事件                           | -      |
| animationType | 图标需要执行的动画效果，支持：rotate           | -      |
| reactScale    | 用户行为响应范围大小，最小值为1                | 1      |
